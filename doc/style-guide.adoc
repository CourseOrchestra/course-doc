== Общий шаблон описания рабочего места

В данном разделе представлено описание полной структуры рабочего места, по которому оформляются все страницы справки.

== Назначение рабочего места (обязательно)

В начале описания каждого рабочего места следует сообщить читателю вводную информацию:

. Цели использования рабочего места
. Кому доступно (нужно ли дополнительное разрешение доступа)
. Дополнительные условия для начала работы в рабочем месте

== Локальное оглавление (опционально)

Если описание рабочего места довольно объёмное (более двух экранов), то пользователю лучше показать его полную структуру для упрощения поиска информации.
